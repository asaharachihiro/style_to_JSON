このプラグインは、Figmaファイルから
variables と styles を JSON形式でエクスポートするためのものです。
本来はToken Studioなどのプラグインの使用が一般的ですが、社内セキュリティ規定上、こうしたアクセス制限のないプラグインが利用できないため、今回このように自作しています。

出力JSONは Style Dictionary での変換を前提にしています。
Figmaで参照できない関係（alpha分離など）は、JSON側で正規化しています。

**--プラグイン実装時の注意--**<br>
実装の際は以下の制限があることがわかりました。（2025/12時点）

- ?. / ?? が使えない
- variables 取得は必ず非同期
- primitives 生成も必ず非同期
